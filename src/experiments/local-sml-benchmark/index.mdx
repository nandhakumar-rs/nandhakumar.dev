<ExperimentOverview>

This experiment compares small language models (SLMs) running locally, with a focus on measuring and benchmarking practical metrics side by side.

The goal is to understand how different SLMs behave under the same local setup — latency, throughput, resource usage, and output quality — so the trade-offs are clear before picking a model for a real workload.

Architecture, tooling, and detailed results will be documented as the experiment progresses.
